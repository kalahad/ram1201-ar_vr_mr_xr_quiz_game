เกมชื่อ "REALITY SCANNER"  มาเป็นเกมที่เล่นทีละข้อแบบ real-time พร้อมกลไกเกมที่ทำให้อยากเล่นซ้ำ:

⏱ ตัวจับเวลา 12 วิ/ข้อ แบบ HUD วงกลม เปลี่ยนสีเขียว→ส้ม→แดงเมื่อใกล้หมดเวลา สร้างความเร่งเร้า
🔥 ระบบสตรีค (combo) ตอบถูกต่อเนื่องยิ่งได้คะแนนโบนัสมากขึ้น มีไอคอนไฟลุกโชว์
💯 คะแนนแบบไดนามิก = คะแนนพื้นฐาน + โบนัสความเร็ว + โบนัสสตรีค (ตอบเร็ว/ต่อเนื่องยิ่งคุ้ม)
🎯 Radar scanner animation หมุนสแกนระหว่างคิด แล้วเด้ง ✅/❌ ตอนตอบ
🔀 สุ่มลำดับคำถามและปุ่มตัวเลือกทุกรอบ เล่นซ้ำแล้วไม่จำเจ
🏅 ระบบยศ S/A/B/C/D ตอนจบเกม พร้อม confetti ฉลองถ้าทำได้ดี
📋 ทบทวนคำตอบ พร้อมคำอธิบายทุกข้อ ช่วยเรื่องการเรียนรู้จริง
🔊 เสียงเอฟเฟกต์ สร้างสดด้วย Web Audio (ไม่ต้องโหลดไฟล์เสียง) เปิด/ปิดได้
🏆 สถิติสูงสุดในรอบเซสชัน กระตุ้นให้อยากเล่นใหม่เพื่อทำลายสถิติตัวเอง
